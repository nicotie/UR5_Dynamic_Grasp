## UR5_Dynamic_Grasp

## Overview
This project demonstrates tracking and grasping of dynamic objects using the UR5 robot arm in MuJoCo, utilizing `anygrasp_sdk` and `pbvs visual servoing`

## Installation
1. Install and compile conda environment following [graspnet-baseline](https://github.com/graspnet/graspnet-baseline), [anygrasp_sdk](https://github.com/graspnet/anygrasp_sdk.git) and [manipulator_grasp](https://gitee.com/chaomingsanhua/manipulator_grasp.git).
2. Copy `anygrasp checkpoint` to `/ckp` in the repo root path.
3. According to your python version, copy two packages required by anygrasp_sdk (`tracker.so` and `lib_cxx.so`) to the repo root path. Refer to [grasp_tracking](https://github.com/graspnet/anygrasp_sdk/tree/main/grasp_tracking).

## Run demo
```
python rotate.py
python y_only.py
```
